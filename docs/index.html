미용실 예약 시스템

이 프로젝트는 미용실을 위한 간단한 웹 기반 예약 시스템입니다. 특히 나이 든 분들이 쉽게 사용할 수 있도록 큰 폰트와 간단한 UI로 설계되었습니다.

GitHub 저장소 방문

주요 기능

날짜 선택

일주일 단위로 날짜를 표시하며, 일요일은 휴무로 설정됩니다.

시간대별 예약

각 시간대별로 예약 가능 여부를 확인하고 예약할 수 있습니다.

서비스 선택

컷(1시간)과 펌(2시간) 서비스를 선택할 수 있습니다.

예약 관리

예약 목록을 확인하고 필요한 경우 취소할 수 있습니다.

설치 방법

GitHub 저장소에서 코드를 클론하고 필요한 패키지를 설치한 후 실행할 수 있습니다. 자세한 내용은 GitHub 저장소의 README를 참조하세요.

기술 스택

이 프로젝트는 Python, Flask, SQLite, HTML/CSS/JavaScript를 사용하여 개발되었습니다.
